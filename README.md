# Todo CRUD App
A simple todo list app to practice Create, Read, Update, and Delete operations with Supabase.

## Getting Started
1. Clone: `git clone https://github.com/LoisBN/fpp-todo-crud.git`
2. Install: `npm install`
3. Copy `.env.example` to `.env` and add your Supabase credentials
4. Run: `npm run dev`

## Your Tasks
- Create a `todos` table in Supabase
- Implement fetching todos on page load
- Add new todos via a form
- Toggle todo completion status
- Delete todos
- Add error handling for all operations
